# Plan MasCan Puppy APP

## Etapa 1

- App online mínima.
- Carga manual de ventas preparadas.
- Control de pedidos con pistola.

## Etapa 2

- Conectar API MELI.
- Leer ventas del día directamente.
- Descargar etiquetas desde MELI.

## Etapa 3

- Control de carga.
- Persistencia por día.
- Usuarios simples para Manuel y Martín.

## Regla operativa

Manuel no debe recibir claves de MELI. Las credenciales deben quedar como secretos
en Streamlit Cloud o en un servidor controlado por MasCan.
